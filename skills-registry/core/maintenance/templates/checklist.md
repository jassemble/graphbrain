# Maintenance Checklist

## Bug Fix
- [ ] Bug reproduced with failing test
- [ ] Root cause identified (5 whys completed)
- [ ] Fix addresses root cause, not symptom
- [ ] Regression test added and passing
- [ ] Same pattern checked elsewhere in codebase
- [ ] Changelog updated
- [ ] Postmortem written (P0/P1 only)

## Compound Learning
- [ ] Pattern recorded in .ctx/patterns.md (if systemic)
- [ ] Decision updated in .ctx/decisions/ (if assumption changed)
- [ ] Module page updated in .ctx/modules/ (if behavior changed)
- [ ] Lint rule or test added (if preventable)

## Tech Debt
- [ ] Debt item logged with estimated effort
- [ ] Categorized: safety / improvement / cleanup
- [ ] Dependencies identified
